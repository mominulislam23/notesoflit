== Description ==
Lifestyle Blog WordPress Theme is a  child theme for Lifestyle Magazine WordPress Theme, that is perfect for lifestyle & travel bloggers, style guides, personal blogging / portfolio, feminine, traveller, journal, music band & singers, food photography & photographers, writers, fashion designer, interior designers, photo gallery, wedding, ecommerce, chic female blogs and more. Even Though the theme is packed with loads of features, it is very light weight. It is very easy to use and customize with live preview. It supports Woocommerce plugin and is SEO optimized.


== Frequently Asked Questions ==

	= How to install the theme =
		* In your admin panel, go to Appearance > Themes and click the Add New button.
		* Click Upload and Choose File, then select the lifestyle-blog.zip file. Click Install Now.
		* Click Activate to use your new theme right away.
		* Go to Customiser -> Home Page Setting -> Select - Static Front Page
		* Create categories, pages and post
		* Select category needed for the homepage
